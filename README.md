# 小红书Cookie获取器

一个简单易用的Chrome扩展，用于获取小红书网站的Cookie信息。

## 功能特点

- 一键获取小红书网站Cookie
- 支持自定义域名规则
- 提供Cookie模板管理
- 自动复制到剪贴板
- 友好的用户界面

## 安装步骤

1. 下载项目代码
2. 打开Chrome浏览器，进入扩展程序页面（chrome://extensions/）
3. 开启开发者模式
4. 点击「加载已解压的扩展程序」
5. 选择项目文件夹

## 使用说明

### 快速获取Cookie

1. 访问小红书网站
2. 点击扩展图标
3. Cookie将自动复制到剪贴板

### 自定义域名规则

1. 点击扩展图标
2. 点击「设置」按钮
3. 在「域名规则」标签页添加新规则
4. 验证并保存规则

### Cookie模板管理

1. 进入设置页面
2. 切换到「Cookie模板」标签
3. 输入域名和Cookie字符串
4. 点击「分析」创建模板

## 主要功能

### Cookie获取
- 支持一键获取当前网站Cookie
- 自动过滤并格式化Cookie信息
- 即时复制到剪贴板

### 域名管理
- 默认支持小红书域名
- 支持添加自定义域名规则
- 正则表达式匹配支持

### 模板系统
- Cookie模板创建和管理
- 自动识别必需字段
- 模板验证功能

## 技术栈

- Chrome Extension API
- JavaScript
- TailwindCSS

## 注意事项

- 请确保已登录小红书网站
- 使用前需开启扩展的Cookie访问权限
- 建议定期更新Cookie模板
